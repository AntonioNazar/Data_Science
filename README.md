# 1. Sobre o que se tratam esses projetos?
Esses projetos tem um intuito de ser meu portfólio inicial para Ciência de Dados. Grande parte dele está sendo desenvolvido ao longo do programa de formação em Data Science da Oracle. Mas ao longo do tempo será desenvolvidos outros projetos com o intuito de demonstrar minha evolução e tirar certificações, como da IBM.
Baseado nisso, esse repositório só tem fins didáticos e acadêmicos. Para desenvolver conhecimentos nas ferramentas, mas principalmente desenvolver um raciocínio pós manipulação dos dados.
